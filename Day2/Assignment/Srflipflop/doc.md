An SR Flip-Flop (Set-Reset Flip-Flop) is a basic sequential logic circuit used to store one bit of information. It has two control inputs, S (Set) and R (Reset), and two outputs, Q and Q̅. When the Set input is activated, the flip-flop stores a logic 1, and when the Reset input is activated, it stores a logic 0. If both inputs are inactive, the previous state is retained, providing memory functionality. The SR Flip-Flop serves as the foundation for many other flip-flop designs and is widely used in digital systems for data storage and state control. The design is implemented in Verilog HDL and verified through simulation in Vivado, demonstrating correct set, reset, and hold operations.
