A Universal Shift Register is a sequential circuit capable of performing multiple data transfer operations within a single design. The 4-bit Universal Shift Register implemented in Verilog HDL supports four operating modes selected by a 2-bit control signal. In Mode 00, it functions as a Serial-In Serial-Out (SISO) register, shifting data serially through the register. In Mode 01, it operates as a Serial-In Parallel-Out (SIPO) register, accepting serial data and making it available simultaneously at parallel outputs. In Mode 10, it works as a Parallel-In Serial-Out (PISO) register, loading parallel data and shifting it out serially. In Mode 11, it acts as a Parallel-In Parallel-Out (PIPO) register, allowing parallel loading and retrieval of data. The design is implemented using Verilog HDL and verified through simulation in Vivado, demonstrating correct operation in all four modes along with proper data shifting, loading, and output generation
