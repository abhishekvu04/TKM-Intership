The D Flip Flop Consists a single data input(D), a clock input(CLK),and two outputs: Q and Q' (the complement of Q.Case 1 (PR=CLR=0):This conditions is represents as invalid state where both PR(present) and CLR(clear) inputs are inactive. Case 2 (PR=0 and CLR=1):This state is set state in which PR is inactive (0) and CLR is active(1) and the output Q is set to 1. Case 3 (PR=1 and CLR=0):This state is reset state in which PR is active (1) and CLR is inactive (0) and the complementary output Q' is set to 1. Case 4 (PR=CLR=1):In This state the flip flop behaves as normal, both PR and CLR inputs are active(1)
